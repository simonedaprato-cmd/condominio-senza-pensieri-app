# Condominio Senza Pensieri App

Struttura pronta per Vite + React + Tailwind + Supabase.

## 1. Installa dipendenze

```bash
npm install
```

## 2. Configura Supabase

Copia `.env.example` in `.env` e inserisci i valori reali:

```bash
cp .env.example .env
```

## 3. Avvia in locale

```bash
npm run dev
```

## 4. Build

```bash
npm run build
```

## Logo

Il logo è già incluso in `public/logo-condominio-senza-pensieri.png`.

## Note

- Se le variabili Supabase non sono configurate, l'app funziona in modalità demo locale.
- Per il login via email, in Supabase imposta correttamente `Site URL` e `Redirect URLs`.
